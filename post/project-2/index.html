Advancing Market Price Prediction with Machine Learning Models: A Research Study

February 22, 2022

Reasearch on Market price prediction by utilizing machine learning is my research project during the internship work at PT GIT Solution as a Data Analyst. The objective is to find the most accurate model to predict stock price movements and to analyze any available insight from the data. The flowchart below is the flowchart of the proposed pipeline for the model.

My main responsibilites for this projects include:

  • Literature review
  • Data scrapping and labeling Data preparation and pre-processing
  • Test and evaluate proposed models
  • Create visualization and reports

The proposed model uses historical analysis based on data collected from open source data combined with sentiment analysis using the NLP lexicon model. However, because the lexicon model for sentiment analysis was too complicated and the internship period was limited, the model development was continued only based on historical data analysis.

I used 4 stock tickers that were sampled for the research: BBCA, MDKA, SAME, and HRUM. The four stock data are collected and processed using the following workflow. Exploratory data analysis, data preparation, data modeling and evaluation. I compared the performance of 4 different algorithms used at the modeling stage:

  • FB - Prophet
  • Multi-Layer Perceptrons
  • Linear Regression
  • Long-short Term Memory

The evaluation results below show that the LR model has the highest average score.

Research conclusion:

  • Machine Learning models are quite effective when predicting short-term price changes, but they are not good for predicting long-term price changes, because the data used to predict are the results of previous predictions.

  • Based on the evaluation of the forecasting model, the error obtained is still quite significant and can certainly greatly affect the level of confidence in decision making.

  • If we look at the movement of the stock, namely up or down, the model is good enough to predict whether the stock will go up or down with an accuracy of 75%.
